# 硬件与权限确认 v1

## 已确认硬件

### 1. 标签打印机

- 型号：`得力 DL-720C`
- 用途：
  - 商品 barcode 标签
  - 后续可扩展打印调拨单

### 2. 标签尺寸

- 服装：`60mm x 40mm`
- 百货：`40mm x 30mm`

### 3. 收银硬件

- 一台 Windows 收银机
- 一台 POS 配套 3D 扫码枪

## 标签内容规则

- 展示给顾客和收银员的内容：
  - `barcode`
  - `品名`
- 当前条码标准：
  - `Code128`
- 不在热敏标签上打印：
  - 仓库货架位
  - 门店货架位
- 第一版标签排版默认：
  - 品名置顶
  - 条码置中
  - 条码文本置底
  - 品名超长时自动截断到 `1-2` 行

## 在营门店

- `UTAWALA`
- `PAIPLINE`
- `KINNO`

## 权限角色建议

### 仓库员

- 录入商品
- 录入收货
- 打印标签
- 按调拨单拣货

### 仓库主管

- 审核调拨单
- 查看仓库库存
- 查看打印记录

### 店长

- 签收调拨
- 提交差异
- 编辑门店货架位
- 查看本店库存与销售

### 收银员

- POS 销售
- 查看当前可售商品

### 区域主管

- 审批差异库存调整
- 审核门店关键异常

### 管理员

- 门店管理
- 角色管理
- 系统参数管理

## POS 终端方案

- 第一版直接使用现有 Windows 收银机运行网页 POS
- 扫码输入来自现有 3D 扫码枪
- 后续如果需要更稳的本地打印和硬件控制，再补桌面壳程序
