## BUG-005：POS 页面不适配 1366 × 768 收银一体机

编号：BUG-005  
标题：POS 页面在 1366 × 768 收银一体机上显示不完整  
环境：Staging  
账号 / 角色：Cashier  
页面路径：POS 页面  

问题现象：  
POS 页面没有完整适配 1366 × 768 的收银一体机屏幕。  
页面需要向下滚动很多才能看到完整内容，不利于收银员快速操作。

1. 打开 staging POS 页面。  
2. 使用 cashier 账号登录。  
3. 在 1366 × 768 收银一体机屏幕上查看 POS 页面。

POS 页面必须适配 1366 × 768 收银一体机。  
收银员应能在一屏内完成主要 POS 操作。

浏览器顶部栏和 Windows 底部任务栏存在时，也不能遮挡核心操作按钮。

实际结果：  
页面过高，需要下拉很多才能显示完整。
